# Personal Portfolio Website

A modern and responsive personal portfolio website built using HTML, CSS, and JavaScript.

## 👨‍💻 About

This portfolio belongs to Taimour Sultan, a Software Engineering student and Web Developer. The website showcases personal information, skills, education, experience, projects, blog section, and contact details. 

## 🚀 Features

- Responsive Design
- About Me Section
- Resume & Education Timeline
- Skills Progress Bars
- Portfolio Projects
- Blog Section
- Testimonials Modal
- Contact Form
- Social Media Links
- Interactive Navigation Menu

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

The project uses JavaScript for:

- Sidebar Toggle
- Portfolio Filtering
- Modal Popups
- Contact Form Validation
- Page Navigation.
